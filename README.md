# MIMUW_WWW_ZAD2

## How to compile server
```
npm run tsc
```

## How to run server
```
npm run start
```

## How to run tests
Make sure server is down.

Test console
```
npm run beforeTest
```

Server console
```
npm run start
```

Test console
```
npm run test
```

Server console
```
Ctrl + C // close server
```
